O iShorts é uma ferramenta **100% responsiva**. Isso significa que você não precisa criar versões separadas dos vídeos: o sistema detecta automaticamente o dispositivo do cliente e ajusta o layout para a melhor experiência.

## Experiência Mobile (Celular)

No celular, o foco é a imersão total, replicando a experiência de aplicativos de redes sociais que o cliente já está acostumado.

* **Navegação:** Otimizada para o toque (touch). Em carrosséis, o cliente desliza para o lado (*swipe*).
* **Visualização:** Ao clicar, o vídeo abre em tela cheia (Full Screen).
* **Interação:** Os botões de curtir, compartilhar e o **Card do Produto** ficam posicionados estrategicamente na parte inferior para facilitar o clique com o polegar.

## Experiência Desktop (Computador)

No computador, o layout se adapta para aproveitar a largura da tela.

* **Navegação:** O cliente utiliza o mouse. Em carrosséis, setas de navegação aparecem nas laterais.
* **Visualização:** O vídeo abre em um modal (janela sobreposta) centralizado, mantendo o cliente na mesma página sem perder o contexto da navegação.
